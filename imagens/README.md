# Imagens da landing

Esta pasta guarda os **screenshots da extensão** que vão aparecer na página de vendas.

## Quais imagens preciso

Recomendado capturar (não obrigatório, mas ajuda muito a vender):

1. **`calculadora.png`** — print da calculadora completa, com Clássico e Premium preenchidos.
   - Tamanho recomendado: 1200×800 ou maior.
   - Capturar dentro do painel real, mostrando os dois cards de resultado.

2. **`anuncio-detectado.png`** — print mostrando o card "ANÚNCIO ATUAL" preenchido com dados reais.
   - Tamanho recomendado: 1000×500.

3. **`botao-flutuante.png`** — print da página do Mercado Livre com o botão "Ferramentas" no canto.
   - Tamanho recomendado: 1400×900.

4. **`tela-login.png`** — print da tela de login (passo do email e/ou do código).
   - Tamanho recomendado: 800×600.

## Como tirar prints no Windows

1. Aperte a tecla **PrintScreen** (PrtSc) ou **Windows + Shift + S** pra recortar uma área.
2. Cole no Paint, ajuste, salve como **PNG**.
3. Coloque o arquivo nesta pasta com o nome sugerido.

## Como pôr na landing depois

Quando você adicionar as imagens aqui, me avise e eu insiro elas nos lugares certos do `index.html` e ajusto o CSS pra elas ficarem bonitas.
